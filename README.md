# Features

# Menu Ordering:

Browse the menu, select dishes, and place orders.

View details such as dish name, description, price, and availability.

Add items to the cart and proceed to checkout.

# Table Reservation:

Reserve a table for a specific date and time.

Specify the number of guests.

Receive confirmation and reminders via email or SMS.


# Technologies Used

# Frontend:

React: Building the user interface.

Redux: State management.

Axios: Handling API requests.

React Router: Navigation.

# Backend:

Express: Creating RESTful APIs.

MongoDB: Storing reservation details
.
Mongoose: Object Data Modeling (ODM).
